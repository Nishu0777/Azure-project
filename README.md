# Hello Azure App

## Objective
A minimal Python Flask app deployed on Azure App Service.

## Features
- Home endpoint (/)
- Health check endpoint (/health)

## Deployment
- Hosted using Azure App Service
- Connected via GitHub for CI/CD

## How to run locally
pip install -r requirements.txt
python app.py
